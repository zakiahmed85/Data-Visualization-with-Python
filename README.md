## âš™ï¸ Getting Started

### Prerequisites

To run this project locally, you will need **Python** installed. It is recommended to use a virtual environment.

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/zakiahmed85/Data-Visualization-with-Python
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Data-Visualization-with-Python
    ```
3.  **Install the required libraries** (using `pip`):
    ```bash
    pip install pandas numpy matplotlib seaborn jupyter
    ```

### Usage

To explore the project, launch the Jupyter Notebook server:

```bash
jupyter notebook Data_Visualization_with_Python.ipynb
```
